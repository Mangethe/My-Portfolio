# G. Nqobile Shabangu - Data Scientist Portfolio

A professional, multi-page portfolio website designed to showcase the work, skills, and experience of a Data Scientist and Platform Engineer. This project demonstrates responsive web design, modern CSS animations, and a clean, accessible user interface.

## 🚀 Features

- **Multi-Page Layout**: Distinct sections for Home, Projects, Contact, and Résumé.
- **Responsive Design**: Fully adaptable layout that works seamlessly on mobile, tablet, and desktop devices.
- **Accessibility (a11y)**:
  - ARIA attributes for screen reader compatibility.
  - Keyboard-accessible navigation and interactive elements.
  - Semantic HTML structure.
- **Interactive UI**:
  - Smooth scrolling navigation.
  - Mobile-friendly slide-in side menu.
  - 3D Flip cards for displaying technical skills (supports hover & focus).
  - Entrance animations for text and elements.
- **Dark Luxury Theme**: A high-contrast, professional color palette using CSS variables for consistency.
- **Résumé View**: Dedicated HTML view for the CV with a PDF download option.
- **Contact Form**: Styled contact form structure ready for backend integration.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and accessibility.
- **CSS3**:
  - Flexbox & Grid for layout.
  - CSS Variables for theming.
  - Keyframe animations and transitions.
  - Media queries for responsiveness.
- **JavaScript**:
  - Mobile menu toggling logic.
  - Event handling for navigation.
